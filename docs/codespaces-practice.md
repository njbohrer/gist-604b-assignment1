## Understanding Git Workflow
Explain in 2–3 sentences the difference between:
- Local workspace (Codespace)
- Remote repository (GitHub.com)
The local workspace (Codespace) is where I make changes and run my code. The remote repository (GitHub.com) is where the project is stored online. I make changes locally, then commit and push them so they show up on GitHub.
